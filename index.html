World Wide Web - The Basics

The World Wide Web

The World Wide Web is a collection of webpages, images and interactive content (Web applications) connected together by links (Hyperlinks - commonly abbreviated to link) .

Key Components of the Web:

  • The Browser on My Computer
  • The Webserver - located 'out there' - on the Cloud

These two elements communicate via HTTP (Hyper Text Transfer Protocol). The browser sends a request and the webserver returns the requested webpage.

How does the webserver know which page to the browser?

The browser requests a page by clicking on a Hyperlink which contains a reference to a URL (Uniorm Resource Locator) which is unique to each page on the web.

Webpage Overview

Webpages are documents of type HTML (HyperText Markup Language).

HTML comprises four key types of things:

  1. text (what you see)
  2. markup (what it looks like)
  3. references to other documents (video and images))
  4. links to other pages
The HTML Element

The contents of an HTML Document is made up of Elements.

All Elements are rectangluar in shape.

HTML <tags> are used to markup these Elements.

These <tags> tell the browser how to handle the individual elements of the webpage.

There are two major types or elements:

  • Inline Elements
    Simple in nature and have only content and occupy only the space they need.

  • Block Elements
    Contained within a virtual box and have height and width attributes as well as other potential. By default they occupy the full width of the webpage.



Most Elements are written with an opening tag, content and a closing tag.
eg. <tag>content</tag>..

In addition, some elements (like images and links) can have (an) attribute(s) which have names and values.
eg. <tag attributeName="value">content</tag>

The are also a number of tags which are referred to as void tags which don't have either content or a closing tag.
Examples:
The image element: <img src="url" alt="text" >
(note: the 'alt' attribute is not compulsory but is important as it is what will be displayed if there is an error/break in the link to the image url. It is also helpful for blind users.)

The Line Break element: <br>
